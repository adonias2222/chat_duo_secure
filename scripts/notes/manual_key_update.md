# Atualização planejada: chave compartilhada local

Esta atualização corrige o problema de mensagens aparecendo como `Não foi possível descriptografar` após reinstalar ou atualizar o APK.

Mudança prevista:

- Adicionar uma chave local do chat.
- Pedir a mesma chave nos dois celulares ao abrir a conversa.
- Salvar a chave apenas no armazenamento seguro do aparelho.
- Usar essa chave para mensagens novas, fotos, vídeos e áudios.
- Manter tentativa de leitura das mensagens antigas com a chave anterior.

Observação: mensagens antigas criptografadas com chaves que foram perdidas não podem ser recuperadas. Depois que os dois celulares usarem a mesma chave, as mensagens novas devem funcionar normalmente.
